WELCOME TO MY NEWS APP

To all those readers who want to skim through the top news of the day , This is the App for you.

This project uses Google API for extracting the news from various platform and Newspaper3k library for further processing and summerisation.

Initial idea was to use BERT based summerizer but due to lack of processing power on my local system, i used Newspaper3k's in house summerizer.

But to those who have the processing power , i have attached a python file which shows how to use BERT summerizer for the current project.

You just have to replace the code with BERT as shown in my python file (check out in my github repo).
